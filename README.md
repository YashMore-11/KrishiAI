# KrishiAI 🌿
**An AI-powered ecosystem for sustainable agriculture and plant health.**

Agriculture faces massive losses due to undetected crop diseases. Farmers need instant, expert advice at their fingertips, even in areas with low internet connectivity. **KrishiAI** bridges this gap by combining Edge Machine Learning for offline disease detection with Cloud-based Generative AI for expert agricultural consultation.

---

## ✨ Key Features

*   **🔍 Offline Disease Detection (Edge ML):** Scan plant leaves in real-time without an internet connection. The custom-trained model diagnoses healthy and diseased states across Tomatoes, Potatoes, and Corn with high confidence, while filtering out non-plant objects.
*   **🤖 AI Agricultural Assistant:** Powered by a fine-tuned Large Language Model (LLM), the KrishiAI chatbot acts as a virtual agricultural expert, providing contextual, domain-specific remedies and farming advice.
*   **⛅ Real-Time Weather Integration:** Fetches localized weather data to help farmers make informed decisions about watering, pesticide application, and harvesting.
*   **📱 Cross-Platform UI:** A clean, intuitive, and highly responsive mobile interface built specifically for ease of use in the field.

---

## 🛠️ Tech Stack

*   **Frontend Framework:** Flutter (Dart)
*   **Machine Learning (Vision):** TensorFlow Lite (MobileNet architecture)
*   **Generative AI (Chatbot):** Google Gemini 2.5 Flash API
*   **External APIs:** Open-Meteo REST API (Weather)
*   **Methodology:** Agile Iterative Prototyping
